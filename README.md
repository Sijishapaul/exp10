AIM

To study and implement the Pandas library in Python for creating and manipulating Series and DataFrames, and to perform basic data handling and analysis operations.

THEORY

Pandas is a powerful Python library used for data manipulation and analysis. It provides two main data structures: Series, which is a one-dimensional labeled array, and DataFrame, which is a two-dimensional tabular structure consisting of rows and columns. Pandas allows efficient storage, retrieval, filtering, and analysis of structured data.

Data can be accessed using indexing and column selection methods. Pandas also provides built-in statistical and descriptive functions that help summarize datasets quickly. Due to its simplicity and efficiency, Pandas is widely used in data science, machine learning, and data analysis applications.

Series() – Creates a one-dimensional labeled array capable of holding different data types.

DataFrame() – Creates a two-dimensional table-like data structure with rows and columns.

head() – Displays the first few rows of a DataFrame.

tail() – Displays the last few rows of a DataFrame.

describe() – Generates statistical summary information of numerical columns.

mean() – Calculates the average value of a column or dataset.

sum() – Computes the total of values in a column or DataFrame.

max() – Returns the maximum value in the dataset.

min() – Returns the minimum value in the dataset.

columns – Displays the column labels of a DataFrame.

index – Shows the row labels of a DataFrame.

loc[] – Accesses data using row and column labels.

iloc[] – Accesses data using integer index positions.
